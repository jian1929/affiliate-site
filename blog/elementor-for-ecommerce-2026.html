Setting Up Elementor with WooCommerce

1 Install Required Plugins

You'll need three components:

  • WordPress installed
  • WooCommerce plugin (free)
  • Elementor plugin (free)
  • Elementor Pro (paid - required for e-commerce features)

2 Configure WooCommerce

Set up your store basics:

  • Add your products with images and descriptions
  • Configure payment gateways (Stripe, PayPal, etc.)
  • Set up shipping zones and rates
  • Configure tax settings

3 Install WooCommerce Elementor Widgets

Elementor automatically detects WooCommerce and adds specialized widgets to your panel. Look for the "WooCommerce" section in the Elementor widget panel.

Essential E-commerce Widgets

Product Grid Widget

Display your products in beautiful grid layouts. Customize:

  • Number of products to display
  • Columns (1-6)
  • Product image size and hover effects
  • Add to cart button styling
  • Price display options
  • Sale badge styling

Product Carousel Widget

Showcase products in a sliding carousel. Perfect for:

  • Featured products section
  • Best sellers display
  • New arrivals showcase
  • Related products

Product Categories Widget

Display product categories with images and descriptions. Great for creating a visual navigation menu for your product catalog.

Add to Cart Widget

Add products to cart from any page. Customize button text, styling, and what information to display (price, quantity selector, options).

Designing Product Pages with Elementor

Product pages are crucial for conversions. Elementor gives you complete control over every aspect:

Creating Custom Product Templates

  1. Go to Templates → Theme Builder
  2. Click Add New and select Single Product
  3. Choose from pre-built templates or start blank
  4. Add Elementor widgets to build your layout
  5. Use WooCommerce widgets: Product Title, Price, Image, Rating, Related Products
  6. Set conditions (all products or specific categories)

Product Page Widgets Available

  • Product Title: Customizable heading with breadcrumb support
  • Product Price: Regular and sale price styling
  • Product Image: Main image with gallery thumbnails
  • Product Rating: Star ratings with review count
  • Add to Cart: Quantity, variations, and button styling
  • Product Short Description: Styled excerpt text
  • Product Tabs: Custom tabs for description, reviews, FAQs
  • Related Products: Cross-sell and upsell displays
  • Product Data: SKU, stock status, categories, tags

Building Shop and Archive Pages

Create custom templates for your shop page and product archives:

Custom Shop Page Template

  1. Go to Templates → Theme Builder
  2. Add new template of type Products Archive
  3. Include: Page Title, Product Grid, Pagination
  4. Style to match your brand
  5. Set as your default shop page

Checkout Customization

Elementor Pro's WooCommerce widgets also extend to checkout customization. You can redesign:

  • Checkout page header and footer
  • Order confirmation page
  • Cart page layout
  • Empty cart state

Note: For full checkout field customization, consider Elementor Hello Theme + WooCommerce Checkout Field Editor for maximum control over checkout form fields.

Essential Addons for E-commerce

Enhance your Elementor WooCommerce store with these recommended addons:

  • WooCommerce Additional Variation Images: Show multiple images per product variation
  • WooCommerce Product Gallery Slider: Add smooth gallery transitions
  • WooCommerce Checkout Field Editor: Customize checkout fields
  • WooCommerce Catalog Mode: Disable cart for catalog-only stores
  • WooCommerce Quick View: Product quick view without page navigation
  • WooCommerce Wishlist: Let customers save products for later

Performance Optimization

Keep your e-commerce store fast with these tips:

  • Optimize images: Use WebP format and lazy loading
  • Minimize plugins: Only use essential addons
  • Enable caching: Use a quality caching plugin
  • Use CDN: Speed up global delivery
  • Lazy load: Only load products below the fold when needed
  • Database cleanup: Regularly clean transients and revisions

Mobile Optimization

Over 60% of e-commerce traffic comes from mobile devices. Elementor makes mobile optimization easy:

  • Responsive editing: Edit each breakpoint separately
  • Stack widgets: Automatic stacking for mobile layouts
  • Touch-friendly: Larger tap targets for buttons
  • Simplified navigation: Mobile hamburger menus
  • Fast checkout: Optimized checkout flow for small screens

Conversion Optimization Tips

Key Insight: A well-designed product page can increase conversions by 20-40%. Invest time in your product page design.

  • High-quality images: Multiple angles, zoom capability, lifestyle shots
  • Clear CTAs: Prominent "Add to Cart" buttons with contrasting colors
  • Trust signals: Display security badges, reviews, guarantees
  • Social proof: Customer reviews, ratings, testimonials
  • Scarcity: "Only 3 left" messages for urgency
  • Cross-sells: "You might also like" product suggestions
  • Clear pricing: No hidden fees, show total upfront
