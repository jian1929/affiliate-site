What is Motion?

Motion (formerly Framer Motion) is the de facto standard for React animations. It provides production-ready animation primitives that are both powerful and easy to use, making complex animations accessible to React developers.

Key Features

  • Motion Values: Physics-based animation values
  • Gestures: Drag, hover, pan, pinch support
  • Layout Animations: Automatic layout transitions
  • Variants: Orchestrate multi-step animations
  • Scroll Animations: Scroll-linked motion
  • Exit Animations: Animate on component unmount
  • Shared Layout: Animate between components
  • Server Components: SSR compatible
