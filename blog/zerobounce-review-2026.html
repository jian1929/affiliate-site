How ZeroBounce Works

ZeroBounce uses a multi-layer verification process that goes far beyond simple syntax checking:


  1. Syntax Validation - Checks email format and structure
  2. Domain/MX Validation - Verifies the mail server exists
  3. SMTP Verification - Connects to MX servers to confirm mailbox existence
  4. Catch-All Detection - Identifies accept-all servers
  5. Toxic Domain Check - Flags spam traps and abuse emails
  6. Role-Based Detection - Identifies admin@, info@, support@ addresses
  7. Final Disposition - Classifies as valid, invalid, catch-all, or unknown
