What is Supabase?

Supabase is an open-source Firebase alternative that provides a complete backend-as-a-service solution. Built on PostgreSQL, it offers real-time databases, authentication, storage, and serverless functions—all with a generous free tier and transparent pricing.

Key Features

  • PostgreSQL Database: Full PostgreSQL with real-time subscriptions
  • Authentication: Email/password, OAuth, SSO, and magic links
  • Real-time: Listen to database changes via WebSockets
  • Storage: Large file storage with CDN and image transformations
  • Edge Functions: Serverless TypeScript functions at the edge
  • Auto-generated APIs: REST and GraphQL from your schema

  • Self-hosted Option: Deploy on your own infrastructure
