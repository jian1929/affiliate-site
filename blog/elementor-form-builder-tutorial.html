Creating Your First Form

1 Add the Form Widget

Open any page in Elementor editor. Find the Form widget in the widget panel (search for "form" or look under "Site Elements"). Drag it onto your page.

2 Configure Form Name

In the widget settings, give your form a unique name. This helps you identify forms when setting up integrations and reviewing submissions.

3 Add Form Fields

Click "Edit Form" to open the form fields panel. Use the "Add Item" button to add new fields. Elementor offers:

  • Text (single line)
  • Email (with validation)
  • Tel (phone number)
  • Textarea (multiple lines)
  • Number
  • URL
  • Date/Time picker
  • Select dropdown
  • Checkbox
  • Radio buttons
  • Acceptance checkbox
  • File upload
  • Hidden field
  • Step (for multi-step forms)

4 Customize Field Settings

For each field, you can set:

  • Field label
  • Placeholder text
  • Required vs optional
  • Default value
  • CSS classes
  • Validation rules (min/max length, custom patterns)

Form Field Types Explained

Essential Fields

Name Field: Add two text fields - one for "First Name" and one for "Last Name" or use the default Name field.

Email Field: Always validate email fields. This ensures you collect valid email addresses for follow-up.

Phone Field: Set the input type to "tel" for mobile-friendly number input with appropriate keyboards.

Advanced Fields

File Upload: Allow users to upload documents, images, or files. Configure allowed file types and maximum size.

Acceptance Checkbox: Required for GDPR compliance when collecting consent. Link to your privacy policy.

Hidden Fields: Pass hidden data like UTM parameters, page URLs, or user IDs to track form submissions.

Multi-Step Forms

Multi-step forms improve completion rates by breaking long forms into manageable sections.

Creating Multi-Step Forms

  1. Add a "Step" widget between field groups
  2. Configure step labels and icons
  3. Enable "Hide Labels" for cleaner appearance
  4. Set "Show Progress Bar" for better UX
  5. Add conditional logic to skip steps based on previous answers

Spam Protection

Protect your forms from spam submissions with Elementor's built-in security features:

  • Akismet Integration: Automatically filters spam submissions using Akismet's proven technology.
  • honeypot Fields: Hidden fields that bots fill out but humans don't see, catching automated spam.
  • Google reCAPTCHA: Add invisible or checkbox reCAPTCHA to your forms.
  • Unique Submit: Prevents double submissions from impatient users.

Managing Form Submissions

View and manage all form submissions from your WordPress dashboard:

  1. Go to Elementor → Submissions
  2. View all form entries in a clean table
  3. Filter by form name, date, or status
  4. Export submissions to CSV
  5. Mark entries as read/unread
  6. Delete unwanted submissions

Best Practices

  • Keep it short: Only ask for essential information. Each field reduces conversion rates.
  • Clear labels: Use descriptive labels that explain what you need and why.
  • Smart defaults: Pre-fill fields when possible to reduce user effort.
  • Mobile optimization: Test forms on mobile devices for usability.
  • Confirmation message: Show a personalized thank you message after submission.
  • Email notification: Set up admin notifications with relevant submission details.

Troubleshooting Common Issues

Form not sending emails

Check your WordPress mail settings. Many hosts block PHP mail. Consider using a SMTP plugin like WP Mail SMTP.

Integrations not connecting

Ensure your email marketing account is properly connected. Try disconnecting and reconnecting the integration.

Form styling looks broken

Check for theme conflicts. Try switching to a default WordPress theme temporarily to isolate the issue.
