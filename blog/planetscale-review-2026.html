What is Planetscale?

Planetscale is a serverless database platform built on MySQL. Founded by former MySQL engineers at YouTube, it brings enterprise-grade MySQL to developers with features like database branching, auto-scaling, and excellent developer tooling.

Key Features

  • Database Branching: Git-like workflow for your database schema
  • Serverless Architecture: Auto-scales from zero to millions of queries
  • Vitess Powered: The same technology used by YouTube
  • Read Replicas: Instant read scaling with no configuration
  • Deploy Requests: Review and approve schema changes
  • Non-blocking ALTER: Change schemas without downtime
  • Row-level security: Filter-based access control
  • HQoS (Quality of Service): Prioritize critical queries
